accessing

verb ˈæksesɪŋ
accessing is the present continuous tense of access

  1. to access a computer file is to find and be able to use it
  2. to see or use information on a computer
  3. to be able to reach or use something
  4. to approach or enter a place

Root word: access (verb)

Related roots: access (noun)

Related words: accesses (verb), accessed (verb)
